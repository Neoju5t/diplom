Архитектура решения

Проект включает следующие компоненты:

  • Веб-серверы NGINX (2 экземпляра)
  • Балансировщик нагрузки Yandex ALB
  • Мониторинг: Zabbix
  • Сбор логов: Elasticsearch + Kibana
  • Резервное копирование: Snapshot дисков
